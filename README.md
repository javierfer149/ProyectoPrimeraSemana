# Proyecto Primera Semana - ALVEARIUM VR

Este es mi primer proyecto creado con React y Vite durante mi primera semana de prácticas. A continuación, defino los conceptos clave que he aprendido:

## ¿Qué es Node?
Node.js es el programa que permite usar herramientas modernas de desarrollo web.

## ¿Qué es npm?
Es el sistema que permite instalar herramientas adicionales dentro de un proyecto.

## Cómo ejecutar el proyecto
Para hacer funcionar este proyecto en tu propio ordenador, debes abrir una terminal en la carpeta del proyecto y ejecutar el siguiente comando:

`npm run dev`

Esto arrancará un servidor de desarrollo local y te proporcionará una ruta (En este caso `http://localhost:5173/`) que puedes abrir en cualquier navegador web.

## ¿Qué es un componente?
Un componente es una parte independiente de la aplicación, como por ejemplo un botón o una tarjeta con información.Toda aplicación web creada con React se construye combinando estos componentes.
